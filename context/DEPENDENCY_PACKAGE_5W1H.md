# Identifikasi Dependency/Package Laravel Proyek PBL ETC Planet

Dokumen ini mengidentifikasi dependency/package yang digunakan atau kemungkinan diperlukan dalam proyek PBL ETC Planet. Pemilihan package mengacu pada kebutuhan project: website company profile, registrasi online, pembayaran, dashboard admin/siswa, role access, testing, styling UI, export Excel rekap siswa, dan generate dokumen rapor.

Status package:

- **Sudah terpasang**: package sudah tercatat di `composer.json` atau `package.json`.
- **Direkomendasikan**: package belum dipasang, tetapi relevan dengan kebutuhan blueprint PBL ETC Planet.

## Catatan Penggunaan dalam Proyek PBL

- Package yang sudah terpasang mendukung fondasi project, development workflow, UI, dan testing.
- Package rekomendasi tidak perlu langsung dipasang jika modulnya belum dikerjakan, tetapi sudah sesuai dengan kebutuhan blueprint project.
- Untuk modul laporan Excel dan rapor, package dokumen harus tetap mengikuti template asli dari folder `context/`, bukan membuat layout bebas.
- Untuk role access, project saat ini dapat memakai role sederhana terlebih dahulu, lalu `spatie/laravel-permission` dapat dipakai jika kebutuhan permission makin detail.
