# Changelog

Semua perubahan penting pada proyek ETC Planet dicatat di file ini. Format mengikuti pola `Unreleased` dan catatan mingguan agar sesuai dengan pengecekan rutin PBL.

## Unreleased

### Added

- Baseline dokumentasi proyek sesuai modul Dokumentasi Proyek Laravel Berbasis PBL.
- Dokumentasi instalasi, fitur, dependency, refactoring, dan rencana GitHub Actions.

### Changed

- README bawaan Laravel diganti menjadi dokumentasi utama proyek ETC Planet.

### Fixed

- Belum ada perbaikan bug yang dicatat pada baseline dokumentasi ini.

### Dependency

- `maatwebsite/excel` sudah tercatat sebagai dependency untuk kebutuhan export Excel.
- Risiko kompatibilitas PHP lokal 8.5 dengan dependency transitif Laravel Excel dicatat pada dokumentasi dependency dan instalasi.

### Refactor

- Baseline refactoring mencatat struktur route terpisah, penggunaan service layer, dan layout/component Blade.

### Docs

- Menambahkan struktur dokumentasi wajib: `docs/installation.md`, `docs/features.md`, `docs/dependency.md`, `docs/refactoring.md`, dan `docs/github-actions.md`.
